# SME 360

**Next Gen Business Management Platform**

SME 360 is a comprehensive solution designed to streamline business operations, from Point of Sale (POS) to Inventory and Customer Management.

## Key Features
*   **Smart POS**: Fast and efficient billing system.
*   **Inventory Control**: Real-time stock tracking and low-stock alerts.
*   **Admin Query System**: Multi-channel support (Web + WhatsApp) for customer queries.
*   **Analytics**: Detailed sales and expense reports.
*   **Role-Based Access**: Secure access for Admins, Managers, and Cashiers.

## Tech Stack
*   **Frontend**: React, Chakra UI, Vite
*   **Backend**: Node.js, Express, MongoDB (Mongoose)

## Getting Started
  
### Prerequisites
*   Node.js (v18+)
*   MongoDB (Local or Atlas)

### Installation

1.  **Clone the repository** (if applicable) or extract files.

2.  **Backend Setup**
    ```bash
    cd backend
    npm install
    # Ensure MongoDB is running
    npm run dev
    ```

3.  **Frontend Setup**
    ```bash
    cd frontend
    npm install
    npm run dev
    ```

4.  **Access the App**
    *   Frontend: `http://localhost:5173`
    *   Backend: `http://localhost:5000`
